# Dyrt Labs — Organization Chart

Interactive org chart for the Dyrt Labs Series A pitch deck and board updates.

## Structure

- **HoldCo** — Dyrt Labs, Inc.: SaaS platform, contracts, G&A shared services
- **OpCo × 10** — Dyrt Operations Co.: standardized physical operations (LA/SoCal Phase 1)
- **ManufacturingCo** — Dyrt Manufacturing Co.: captive hardware supplier

## Hiring phases

| Badge | Meaning |
|-------|---------|
| Current | Active employee |
| Phase 1 | Series A hire |
| Phase 2 | Series B hire |
| OpCo template | Per-site role |
